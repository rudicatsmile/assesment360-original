# Livewire + Flux UI Rules

- Gunakan Livewire v4 (atau latest) + Flux UI components sebanyak mungkin.
- Flux components: FluxTable, FluxButton, FluxModal, FluxInput, FluxSelect, FluxCard, FluxDropdown, FluxBadge, dll.
- Gunakan wire:model, wire:live, wire:debounce untuk real-time.
- Optimistic UI dengan wire:show, wire:text, wire:cloak.
- Dark mode support otomatis (Flux sudah built-in).
- Component naming: PascalCase (TaskList, NotificationBell, dll.).
- Selalu tambahkan pagination, search, dan filter di list component.
